# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- **SEO Global (Admin Master)**: Nova aba de configurações para gerenciar nome do site, metadados gerais e imagem fallback (OG Image).
- **SEO por Projeto**: Campos personalizados de Título, Descrição e Keywords para cada projeto.
- **Social Sharing**: Sistema de compartilhamento com ícones (WhatsApp, LinkedIn, Facebook, Twitter) integrado em perfis e projetos.
- **Google Analytics 4**: Integração oficial via ID de mensuração (Gtag) configurável pelo painel.
- **Analytics Dashboard**: Painel administrativo para visualização de tráfego interno (Views, Visitantes Únicos, Top Páginas).
- **Structured Data (JSON-LD)**: Implementação de Schema.org em páginas de projeto para melhores resultados de busca.
- **Internal Visit Tracking**: Sistema de log de visitas em tempo real para feedback imediato no dashboard.

### Fixed
- **Admin UX**: Correção da visibilidade do ícone de calendário no campo de "Data de Publicação" (corrigido contraste em modo dark).
- **SEO Image Fallback**: Implementada lógica para usar automaticamente a imagem Hero do projeto como miniatura OG caso não haja uma imagem SEO definida.
