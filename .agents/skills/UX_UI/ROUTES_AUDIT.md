# ROUTES_AUDIT.md
Logmedia Portfolio

This document defines the **routing structure, navigation patterns and URL standards** used across the Logmedia Portfolio project.

Its purpose is to ensure:

• consistent navigation  
• predictable URL structure  
• clean routing architecture  
• SEO-friendly URLs  
• optimal user experience  

Agents must follow these rules when analyzing or proposing navigation changes.

---

# ROUTING PRINCIPLES

The routing architecture must prioritize:

• simplicity  
• predictability  
• shallow navigation  
• SEO-friendly URLs  

Users should always understand where they are within the application.

Navigation must feel natural.

---

# PRIMARY ROUTES

The application should maintain a minimal set of primary routes.
